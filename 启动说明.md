# 启动说明

## 快速启动

1. **安装依赖**（如果还没有安装）
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **在浏览器中打开**
   - 开发服务器启动后，访问：`http://localhost:5173`
   - 终端会显示确切的URL

## 其他命令

- **构建生产版本**：`npm run build`
- **预览生产版本**：`npm run preview`

## 故障排除

如果遇到问题：

1. **清除缓存并重新安装**
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

2. **检查Node.js版本**
   ```bash
   node --version  # 应该是 v18 或更高
   ```

3. **检查端口占用**
   - 如果5173端口被占用，Vite会自动使用下一个可用端口
   - 查看终端输出获取实际URL
